## Installing Git

To clone repositories and manage version control, you need **Git installed on your system**.

Git is the tool that allows your computer to communicate with **GitHub repositories**.

Follow the steps below to install Git.

---

### 1️⃣ Download Git

Go to the official Git website:

https://git-scm.com/downloads

The website automatically detects your operating system.

Click the download button for your OS.

Example:

- **Windows → Download for Windows**
- **Mac → Download for macOS**
- **Linux → Installation commands provided on the site**

---

### 2️⃣ Install Git

After downloading the installer:

**Windows**

1. Run the downloaded `.exe` file.
2. Click **Next** through the installation steps.
3. Keep the **default settings** (recommended for beginners).
4. Click **Install**.
5. Finish the installation.

**Mac**

If you have **Homebrew** installed, run:

```
brew install git
```

Or install it using the package from the Git website.

**Linux (Ubuntu / Debian)**

1. sudo apt update
2. sudo apt install git

### 3️⃣ Verify Git Installation

Once installed, open your terminal and run:

```
git --version
```
If Git is installed correctly, you should see something like:

```
git version 2.xx.x
```
